# Census data
This folder contains census data for the three census regions of the UK at constituency level. Links to the original dataset sources are listed below:
 - [England & Wales](https://www.ons.gov.uk/filters/eb18d97d-e7a0-4a00-bc8e-0cc33effcac3/dimensions)
 - [Scotland](https://www.scotlandscensus.gov.uk/)
 - [Northern Ireland](https://build.nisra.gov.uk/en/custom/data?d=PEOPLE&v=PARLCON24&v=RELIGION_BELONG_TO_AGG4)
