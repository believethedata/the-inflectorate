# the-inflectorate

This repo contains all the data and code used to produce the [The Inflectorate](https://believethedata.org/) article.

To replicate the results, run the notebooks in the following order:
 1. Process the geo data by running the notebook in the `geo_data` folder.
 2. Run the notebook in the `constituency_projections` folder to make constituency level projections and create the output files that I used to create the charts for the article (using [Datawrapper](https://www.datawrapper.de/)).

Further details on raw data sources can be found in `README.md` files in the respective folders.

The projections build on the approach outlined in my previous article, [Projecting Belief](https://believethedata.org/just-one-third-of-uk-population-will-be-christian-by-2031/). An overview of these underlying techniques used can be found in a companion article [here](https://believethedata.org/projecting-belief-our-model/).
