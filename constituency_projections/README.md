# Constituency projections
This folder contains a notebook - `constituency_projections.ipynb` - with the main logic for projecting religious identity at the parliamentary constituency level. The best way to understand this logic is to read and/or execute the notebook.

This notebook loads `mps.csv` to join the MP names and party affiliations to constituency projections. This file was dowloaded on 7th June 2026 from [TheyWorkForYou](https://www.theyworkforyou.com/mps/).

This notebook also generates the files used to create the charts in my article. These outputs are written to `../article_charts`.
